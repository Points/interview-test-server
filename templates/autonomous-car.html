Autonomous Car Assignment

Assignment Description:

Points is developing an autonomous vehicle program. It's going well, but we need to iron out a few bugs. The Engineering Team would like to demo our progress to the Product Team, so we need to develop a tool that can take track and travel log data, and then determine whether our autonomous vehicle successfully navigated the track.

Track and travel log data are encoded as JSON. The "track" array contains entries for any position of the track where there is an obstacle in any of the 3 lanes ("a", "b" or "c"). The "travelLog" array contains entries for any position of the track where the car has changed lanes, either to the "left" or "right".

The autonomous car starts at position 0 on lane "b", drives itself to the end of the track and can shift only one lane per position. The simulation should end when the car hits an obstacle, goes out-of-bounds, or when all "track" and "travelLog" entries have been processed.

Front End Instructions

  • Create a simple, yet visually appealing and responsive design.
  • Create a single page web application (a web app that dynamically rewrites the current page with new data) with two screens.
    • First Screen: Display a form, which includes an input field to accept the user's gross income and a submit button that navigates to the second screen.
    • Second Screen: Display how much the user owes in income tax, the effective tax rate and the break down of tax rates and the amounts owed for each applicable tax bracket.
  • The user must be able to navigate back to the first page.
  • Target the latest stable version of Google Chrome

Back End Instructions

  • Implement a simple yet intuitive command line interface that takes in the user's gross income and outputs the user owes in income tax,
    the effective tax rate and the break down of tax rates and the amounts owed for each applicable tax bracket.
